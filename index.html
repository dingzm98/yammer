Search.json

The search resource will return a list of messages, users, topics and groups that match the user’s search query.


users.json

Retrieve users in the current user’s Yammer network. Supports page, sort_by, reverse, and letter parameters.


users/current.json

View data about the current user.



networks/current.json

Returns a list of networks to which the current user has access.



groups/for_user/:User_Id.json

List Yammer groups for a specific user.


users/by_email.json?email=user@domain.com

Search user by email address.


users/in_group/[:id].json

Users in a group. Supports the page parameter.


users/liked_message/[:Message_id].json

Users who have liked a message. Supports the page parameter.
